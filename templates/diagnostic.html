❓ 常见问题

1. Token 登录失败

可能原因：

  • Token 已过期或无效
  • Token 格式不正确
  • 网络连接问题

解决方案：在 www.mefrp.com 重新获取 Token

2. 隧道列表为空

可能原因：

  • 当前账户没有创建任何隧道
  • MEFrp 客户端未连接
  • API 返回格式变化

解决方案：在官网创建隧道，或检查客户端连接状态

3. 节点列表为空

可能原因：

  • 用户组没有分配节点
  • 所有节点都已禁用
  • 网络问题导致获取失败

解决方案：联系 MEFrp 客服分配节点

📖 操作指南

  1. 输入 Token：在上面的输入框中输入您的 MEFrp Token
  2. 测试登录：点击"测试 Token 登录"验证 Token 是否有效
  3. 运行测试：点击"运行全部测试"检查所有 API 功能
  4. 查看日志：在日志区域查看详细的测试结果
  5. 诊断问题：根据失败信息对照常见问题解决方案
