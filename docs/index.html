USAGE

三步 即刻上手

极简部署，开箱即用。无需复杂配置即可享受完整功能。

01

下载并解压

从 GitHub Releases 下载最新版本压缩包，解压到任意目录。

git clone https://github.com/Guoran-11/Guoran-11-MeFrp-GR-Client.git
cd Guoran-11-MeFrp-GR-Client
02

安装依赖

使用 pip 安装项目所需依赖，或直接使用便携版 Python 启动。

pip install -r requirements.txt
# 或双击 稳定启动.bat（自动配置）
03

启动 & 登录

运行 app.py 或 稳定启动.bat，浏览器自动打开 http://127.0.0.1:5001，使用 MEFrp 账号登录即可。

py app.py
# 访问 http://127.0.0.1:5001
🪪

登录

点击右上角"登录"，支持账号密码 / 访问令牌 (Token) 两种方式。

🛣️

创建隧道

进入"创建隧道"页，选择节点、填写本地端口，一键开通。

📊

查看流量

"用户中心" → "流量历史"，切换日/周/月查看趋势。

🛡️

审计日志

在"操作日志"中查看所有敏感操作的完整记录。
