# MEFrp WebUI 启动说明

## 🚀 快速启动

### 方法一：使用启动脚本

双击运行 `launch.bat` 文件即可启动 WebUI。

### 方法二：手动启动

```bash
e:\桌面\1\python_portable\python.exe e:\桌面\1\app.py
```

## 🌐 访问地址

启动成功后，打开浏览器访问：

- **本地访问**：http://127.0.0.1:5000
- **局域网访问**：http://192.168.88.104:5000

## 📋 首次使用流程

### 1. 登录

点击右上角"登录"按钮，进入登录页面。

#### 方式一：Token 登录（推荐）

1. 点击"Token 登录"标签
2. 输入您的访问 Token（Token 不会显示）
3. 点击"Token 登录"按钮

**Token 获取方式**：
- 登录 https://www.mefrp.com
- 进入用户设置
- 找到"访问 Token"或"API Token"
- 复制 Token

#### 方式二：密码登录

1. 输入邮箱和密码
2. 点击"获取验证码"按钮
3. 在弹出窗口完成人机验证
4. 复制验证码到输入框
5. 点击"登录"按钮

### 2. 查看控制面板

登录成功后，您将看到：
- **剩余流量**：您的账户剩余流量
- **隧道数量**：已创建的隧道数量
- **今日签到**：签到状态
- **用户组**：您所属的用户组

### 3. 签到

1. 在控制面板找到"签到"区域
2. 点击"打开验证码"按钮
3. 在弹出窗口完成人机验证
4. 复制验证码到输入框
5. 点击"确认签到"按钮

### 4. 管理隧道

#### 查看隧道

1. 点击左侧菜单"隧道管理"
2. 点击"刷新列表"按钮
3. 查看您的所有隧道

#### 创建隧道

1. 在"创建新隧道"区域
2. 点击"加载节点数据"按钮
3. 填写表单：
   - 选择节点
   - 输入隧道名称
   - 选择隧道类型（TCP/UDP/HTTP/HTTPS）
   - 输入本地 IP 和端口
   - （可选）输入远程端口
4. 点击"创建隧道"按钮

### 5. 查看节点

点击左侧菜单"节点监控"，查看所有可用节点。

## ❓ 常见问题

### Q: 启动失败？

**A**: 检查以下几点：
1. 确保 `python_portable` 目录存在
2. 确保 `app.py` 文件存在
3. 确保端口 5000 未被占用
4. 检查是否有权限运行

### Q: 登录失败？

**A**: 
1. Token 登录：检查 Token 是否正确
2. 密码登录：检查邮箱密码是否正确，验证码是否有效

### Q: 无法创建隧道？

**A**: 检查：
1. 是否已登录
2. 节点是否在线
3. 端口是否在允许范围内
4. 是否超过隧道数量限制

### Q: 验证码无法获取？

**A**: 
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器
